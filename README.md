# 🏥 Hospital Readmission Risk Analysis Dashboard

## 📌 Project Overview

This project analyzes hospital patient data to identify **key factors contributing to 30-day readmissions**. The goal is to uncover trends in patient demographics, diagnoses, and treatment patterns, and provide **data-driven insights to improve patient outcomes and reduce readmission rates**.

---

## 🎯 Objectives

* Identify **high-risk patient groups**
* Analyze **readmission patterns across age, diagnosis, and departments**
* Evaluate **hospital performance**
* Provide **actionable recommendations** to reduce readmissions

---

## 🛠 Tools & Technologies

* **Excel** – Data generation and preprocessing
* **MySQL** – Data cleaning and analysis (queries, subqueries, CASE statements)
* **Power BI** – Data visualization and dashboard creation
* **AI** – Assisted in:

  * Generating realistic healthcare dataset structures
  * Writing optimized SQL queries (beginner → advanced)
  * Creating DAX measures and debugging errors
  * Designing dashboard layout and best practices
  * Improving project documentation and explanations

---

## 🧹 Data Preparation

* Generated synthetic dataset using Excel formulas
* Cleaned missing values and standardized categories
* Created derived fields such as:

  * Age Groups (Young, Adult, Senior)
  * Length of Stay categories
* Imported cleaned dataset into MySQL for analysis

---

## 🧠 SQL Analysis

Performed multi-level SQL analysis:

### 🔰 Beginner

* Total patient count
* Average length of stay
* Gender distribution

### 🟡 Intermediate

* Readmission counts and rates
* Department-level performance
* CASE statements for segmentation

### 🔴 Advanced

* Subqueries to identify high-risk patients
* Above-average stay analysis
* Percentage calculations by diagnosis and department

---

## 📊 Power BI Dashboard

### 🔷 KPI Metrics

* Total Patient Admissions
* Total Readmitted Patients
* Readmission Rate (%)
* Average Length of Stay

---

### 📈 Visualizations

1. **Patient Distribution by Diagnosis**
   → Identifies most common conditions

2. **Readmission Distribution by Age Group**
   → Highlights high-risk age segments

3. **Readmission Trends Over Time**
   → Tracks patterns and seasonality

4. **Patient Distribution by Insurance Type**
   → Shows coverage breakdown

5. **Readmission Rate by Department and Treatment Type (Matrix)**
   → Compares hospital performance across departments

## 🧠 Key Insights

* Senior patients show **higher readmission rates**
* Longer hospital stays are associated with **increased risk**
* Certain diagnoses (e.g., chronic illnesses) drive more readmissions
* Some departments show **higher-than-average readmission rates**

---

## 💡 Recommendations

* Improve **post-discharge follow-ups**
* Focus on **high-risk patient groups**
* Optimize treatment plans for **long stays**
* Enhance care strategies in **underperforming departments**

---

## 🚀 Future Improvements
